### Step 1: Buka Modal Saved Slides

1. Scroll ke bagian bawah modal

#### Option A: Copy ke Clipboard

1. Click button **"Copy Script"**
2. Paste di notes/document untuk editing lebih lanjut

#### Option B: Save ke Database

1. Click button **"Save to DB"**
2. Script akan tersimpan di database untuk referensi

## ğŸ“ Understanding the Script

### Visual Cues Explained

-   `[Close up]` - Shot close up ke wajah atau objek
-   `[Zoom out]` - Zoom out untuk wider shot
-   `[Cut to visual]` - Cut ke visual/grafik/text
-   `[Transition]` - Transisi ke scene berikutnya
-   `[Tunjuk ke teks]` - Gesture pointing ke text
-   `[Pause]` - Jeda natural untuk emphasis
-   `[Wave]` - Wave gesture untuk greeting/goodbye

### Script Structure

```
[Visual Cue] HOOK - 3-5 detik menarik perhatian

INTRO - Perkenalan singkat topik (5-10 detik)

KONTEN UTAMA - Explain poin-poin penting (30-50 detik)
  [Visual cues untuk transisi antar poin]

OUTRO & CTA - Kesimpulan dan call to action (5-10 detik)
```

## ğŸ”§ Troubleshooting

### Script Terlalu Panjang

-   Remove some slides dari carousel
-   Speak faster (but stay clear)
-   Cut redundant transitions

### Script Terlalu Pendek

-   Add more context di setiap poin
-   Add personal story/example
-   Slower, more detailed explanation

### Script Tidak Natural

-   Rewrite dengan kata-kata kamu sendiri
-   Practice sampai feels natural
-   Don't force formality - be yourself

### Generation Failed

-   Check internet connection
-   Try again (AI might be busy)
-   Contact support jika persistent
